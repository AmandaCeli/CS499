# CS 499 5-2 (Enhancement 3)
### The artifact I selected for this enhancement is my CS 340 Project Two Dashboard, which is a web-based dashboard built using Dash and connected to a MongoDB database containing animal shelter data.  This project was originally created in my CS 340 course and was designed to retrieve, display, and filter animal records based on specific rescue categories.  The dashboard allows users to view animal data in a table, interact with visualizations, and display geographic locations on a map.  For this milestone, I focused on enhancing the database functionality of the application to better demonstrate my skills in working with databases and data-driven applications.
### I selected this artifact because it strongly represents my ability to work with databases, particularly MongoDB, and showcases practical skills such as querying, filtering, and retrieving data.  In this enhancement, I improved the way the application interacts with the database by implementing a dynamic query builder that constructs MongoDB queries based on user input.  Instead of relying solely on predefined filters, the updated system now uses MongoDB operators such as $or and $regex to allow flexible keyword searching directly within the database.  Additionally, I enhanced the application by moving filtering logic from the application layer into the database layer, which improves efficiency by reducing unnecessary data processing in Python.  I also added a data cleaning function to ensure that records are consistent and free of missing or improperly formatted values before being displayed.
